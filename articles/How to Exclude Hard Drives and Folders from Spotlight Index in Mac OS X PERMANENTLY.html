How to Exclude Hard Drives and Folders from Spotlight Index in Mac OS X PERMANENTLY?

If you’re a Mac user, you’re probably familiar with Spotlight, the built-in search feature that helps you quickly find files, folders, and applications on your computer. While Spotlight is a great tool for quickly locating items on your Mac, it can also be a bit of a nuisance if it’s indexing files and folders that you don’t need or want to search for. Fortunately, there is a way to exclude hard drives and folders from Spotlight index in Mac OS X permanently.


The first step to excluding hard drives and folders from Spotlight index is to open System Preferences. To do this, click on the Apple icon in the top left corner of your screen and select System Preferences. Once the System Preferences window is open, click on the Spotlight icon. This will open the Spotlight preferences window.


In the Spotlight preferences window, you will see a list of all the hard drives and folders that are currently being indexed by Spotlight. To exclude a hard drive or folder from Spotlight index, simply uncheck the box next to it. Once you have unchecked the box, the hard drive or folder will no longer be indexed by Spotlight.


If you want to exclude a hard drive or folder from Spotlight index permanently, you will need to open the Terminal application. To do this, open the Applications folder and then open the Utilities folder. In the Utilities folder, you will find the Terminal application. Once the Terminal application is open, type in the following command:


sudo mdutil -i off /Volumes/[name of hard drive or folder]


Replace [name of hard drive or folder] with the name of the hard drive or folder that you want to exclude from Spotlight index. Once you have entered the command, press enter. This will permanently exclude the hard drive or folder from Spotlight index.


If you ever want to re-index a hard drive or folder, you can do so by entering the following command in the Terminal application:


sudo mdutil -i on /Volumes/[name of hard drive or folder]


Replace [name of hard drive or folder] with the name of the hard drive or folder that you want to re-index. Once you have entered the command, press enter. This will re-index the hard drive or folder and it will once again be included in Spotlight searches.
