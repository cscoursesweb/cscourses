When moving the "camera" should one move the projection matrix or move the world?

When it comes to moving the “camera” in a 3D environment, there is a debate as to whether one should move the projection matrix or move the world. The answer to this question depends on the type of 3D environment you are working with and the desired effect you are trying to achieve.


If you are working with a 3D environment that is based on a fixed camera, then it is best to move the projection matrix. This will allow you to move the camera around the 3D environment without having to move the world. This is especially useful if you are trying to create a cinematic effect, as it allows you to move the camera around the environment without having to move the objects in the environment.


On the other hand, if you are working with a 3D environment that is based on a moving camera, then it is best to move the world. This will allow you to move the camera around the 3D environment while still keeping the objects in the environment in the same place. This is especially useful if you are trying to create a realistic effect, as it allows you to move the camera around the environment while still keeping the objects in the environment in the same place.
