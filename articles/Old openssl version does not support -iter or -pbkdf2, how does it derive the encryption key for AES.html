Old openssl version does not support -iter or -pbkdf2, how does it derive the encryption key for AES?

The encryption key for AES is derived differently depending on the version of OpenSSL being used. In older versions of OpenSSL, the encryption key is derived using a combination of the cipher algorithm and the message digest algorithm.
