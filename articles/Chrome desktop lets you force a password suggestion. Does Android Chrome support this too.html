Chrome desktop lets you force a password suggestion. Does Android Chrome support this too?

Chrome desktop and Android Chrome are both web browsers developed by Google. While they share many features, there are some differences between the two. One of these differences is the ability to force a password suggestion.


Chrome desktop does allow users to force a password suggestion. This feature is available in the settings menu, under the “Passwords” section. Here, users can enable the “Suggest passwords” option, which will prompt Chrome to suggest a password when creating a new account or changing an existing one.


Unfortunately, Android Chrome does not support this feature. While the browser does offer a password manager, it does not have the ability to suggest passwords. This means that users must manually enter their passwords when creating or changing accounts.


That said, Android Chrome does offer some features that are not available on Chrome desktop. For example, it has a built-in password generator, which can be used to create strong passwords. It also has a “Save passwords” option, which allows users to save their passwords for easy access.
