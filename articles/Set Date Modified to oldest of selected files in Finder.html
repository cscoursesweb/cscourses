Setting the Date Modified to the Oldest of Selected Files in Finder


If you’re a Mac user, you’ve probably used the Finder app to search for and manage files on your computer. It’s a great tool for organizing your files, but sometimes you may need to set the Date Modified to the oldest of the selected files. This can be a tricky task, but with the right steps, you can easily set the Date Modified to the oldest of the selected files in Finder.


The first step is to open the Finder app. You can do this by clicking on the Finder icon in the Dock or by using the Command + Spacebar shortcut. Once the Finder window is open, you can navigate to the folder containing the files you want to modify.


Once you’ve located the folder, select the files you want to modify by clicking on them while holding down the Command key. This will allow you to select multiple files at once. Once you’ve selected the files, right-click on one of them and select “Get Info” from the menu.


This will open a window with information about the selected files. In this window, you’ll see a section labeled “Date Modified”. This is where you can set the Date Modified to the oldest of the selected files. To do this, click on the “Date Modified” field and select the oldest date from the drop-down menu.


Once you’ve selected the oldest date, click on the “OK” button to save your changes. The Date Modified for all of the selected files will now be set to the oldest date you selected.
