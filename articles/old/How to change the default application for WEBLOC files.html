How to change the default application for WEBLOC files?

If you’re a Mac user, you may have noticed that WEBLOC files are becoming increasingly common. WEBLOC files are a type of file that stores a URL link, and they’re used to quickly open a website in your web browser. By default, WEBLOC files are opened with Safari, but you may want to change the default application for WEBLOC files to something else. In this article, we’ll show you how to change the default application for WEBLOC files on your Mac.


First, you’ll need to open the Finder window. To do this, click on the Finder icon in the Dock. Once the Finder window is open, you’ll need to locate the WEBLOC file that you want to change the default application for. Once you’ve located the file, right-click on it and select “Get Info” from the menu.


In the Get Info window, you’ll see a section labeled “Open With”. This section will list the current default application for the WEBLOC file. To change the default application, click on the drop-down menu and select the application that you want to use.


Once you’ve selected the application, click on the “Change All…” button. This will change the default application for all WEBLOC files on your Mac. If you only want to change the default application for a single WEBLOC file, you can click on the “Cancel” button instead.
