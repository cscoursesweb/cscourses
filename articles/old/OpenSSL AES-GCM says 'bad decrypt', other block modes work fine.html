OpenSSL AES-GCM says 'bad decrypt', other block modes work fine?

OpenSSL AES-GCM is a popular encryption algorithm used to secure data in transit. It is a block cipher mode of operation that combines the Advanced Encryption Standard (AES) with the Galois/Counter Mode (GCM).
