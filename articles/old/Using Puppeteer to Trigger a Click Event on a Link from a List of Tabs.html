Using Puppeteer to Trigger a Click Event on a Link from a List of Tabs

Using Puppeteer to Trigger a Click Event on a Link from a List of Tabs on a Page


If you’re looking to automate a task on a web page, Puppeteer is a great tool to use. It’s a Node library that provides a high-level API to control headless Chrome or Chromium over the DevTools Protocol. One of the tasks you can automate with Puppeteer is triggering a click event on a link from a list of tabs on a page. In this article, we’ll walk through how to do this.


First, you’ll need to install Puppeteer. You can do this by running the following command in your terminal:


npm install puppeteer


Once Puppeteer is installed, you’ll need to create a new JavaScript file. This file will contain the code that will trigger the click event.


The first step is to launch a browser instance. You can do this by using the launch() method. This method takes an object as an argument. This object can contain options such as the headless mode, the executable path, and more.


const browser = await puppeteer.launch({ headless: false, executablePath: '/path/to/chrome' });


Next, you’ll need to create a new page. You can do this by using the newPage() method. This method takes an object as an argument. This object can contain options such as the viewport size, the user agent, and more.


const page = await browser.newPage({ viewport: { width: 1920, height: 1080 }, userAgent: 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/80.0.3987.149 Safari/537.36' });


Now that you have a browser instance and a page, you can navigate to the page that contains the list of tabs. You can do this by using the goto() method. This method takes a URL as an argument.


await page.goto('https://example.com/tabs');


Once the page has loaded, you can use the click() method to trigger a click event on the link from the list of tabs. This method takes a selector as an argument. This selector should be the selector of the link you want to click.


await page.click('#tab-link');
