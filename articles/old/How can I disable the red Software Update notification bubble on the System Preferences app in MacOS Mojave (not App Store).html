How can I disable the red Software Update notification bubble on the System Preferences app in MacOS Mojave (not App Store)?

If you’re a Mac user, you’ve probably seen the red Software Update notification bubble on the System Preferences app in MacOS Mojave. This notification is designed to alert you when there are new software updates available for your Mac. While this notification can be helpful, it can also be annoying if you’re not interested in updating your software. Fortunately, there is a way to disable the red Software Update notification bubble on the System Preferences app in MacOS Mojave.


The first step to disabling the red Software Update notification bubble is to open the System Preferences app. To do this, click on the Apple icon in the top left corner of your screen and select “System Preferences” from the drop-down menu. Once the System Preferences app is open, click on the “Software Update” icon. This will open the Software Update window, which will show you any available updates for your Mac.


Once you’re in the Software Update window, you’ll see a checkbox labeled “Show notifications when updates are available”. Uncheck this box to disable the red Software Update notification bubble. Once you’ve unchecked the box, the red Software Update notification bubble will no longer appear on the System Preferences app in MacOS Mojave.


If you ever decide that you want to re-enable the red Software Update notification bubble, simply open the System Preferences app and click on the “Software Update” icon. Then, check the box labeled “Show notifications when updates are available” and the red Software Update notification bubble will reappear.
