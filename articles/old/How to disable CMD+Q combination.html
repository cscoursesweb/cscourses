How to disable CMD+Q combination?

If you’re a Mac user, you’re probably familiar with the CMD+Q combination. This combination is used to quit an application, and it’s a shortcut that many Mac users rely on. However, if you’re looking to disable this combination, it’s actually quite easy to do. In this article, we’ll explain how to disable CMD+Q combination on your Mac.


The first step to disabling the CMD+Q combination is to open the System Preferences. To do this, click on the Apple icon in the top left corner of your screen and select “System Preferences” from the drop-down menu.


Once you’ve opened the System Preferences, click on the “Keyboard” icon. This will open the Keyboard preferences window. In this window, you’ll see a list of keyboard shortcuts. Scroll down until you find the “Quit Application” shortcut. By default, this shortcut is set to CMD+Q.


To disable this shortcut, simply click on the checkbox next to it. This will disable the CMD+Q combination and prevent it from quitting your applications.


If you’d like to set a different shortcut for quitting applications, you can do so by clicking on the “Change Shortcut” button. This will open a window where you can select a new shortcut for quitting applications.


Once you’ve set a new shortcut, click on the “OK” button to save your changes. The new shortcut will now be active and the CMD+Q combination will no longer quit your applications.
