Why does the delete button hovertext say it is a vote, while the button itself doesn't?

The delete button on many websites has a hovertext that says it is a vote, while the button itself does not. This discrepancy can be confusing to users, so it is important to understand why the delete button hovertext says it is a vote, while the button itself does not.


The delete button hovertext is a way to remind users that deleting content is a serious action. By labeling it as a “vote”, the website is emphasizing that the user is making a decision that will affect the content of the website. This is meant to remind users that deleting content is not something to be taken lightly, and that they should think carefully before taking such an action.


The delete button itself does not say it is a vote because it is not a vote in the traditional sense. A vote is usually a decision made by a group of people, while the delete button is a decision made by a single user. Therefore, labeling the delete button as a “vote” would be misleading.
