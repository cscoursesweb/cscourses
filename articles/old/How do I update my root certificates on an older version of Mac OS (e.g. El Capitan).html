How do I update my root certificates on an older version of Mac OS (e.g. El Capitan)?

Updating root certificates on an older version of Mac OS (e.g. El Capitan) is an important security measure that should not be overlooked. Root certificates are digital certificates that are used to verify the identity of a website or server. They are issued by a trusted third-party, such as a Certificate Authority (CA). Without these certificates, your Mac OS may not be able to securely connect to websites or servers.


Fortunately, updating root certificates on an older version of Mac OS (e.g. El Capitan) is relatively straightforward. Here’s how to do it:


1. Open the Keychain Access app. This can be found in the Applications folder.


2. Select the “System Roots” category in the left-hand pane.


3. Select the “Certificates” tab in the right-hand pane.


4. Select the “Update” button at the bottom of the window.


5. Enter your Mac OS administrator password when prompted.


6. Wait for the update to complete. This may take a few minutes.


7. When the update is complete, you should see a message saying “The system root certificates have been updated.”


That’s it! You’ve successfully updated the root certificates on your Mac OS. It’s important to note that this process may need to be repeated periodically in order to ensure that your Mac OS is up-to-date with the latest security certificates.


If you’re having trouble updating the root certificates on your Mac OS, you may need to contact Apple Support for assistance. They can help you troubleshoot any issues you may be having.
