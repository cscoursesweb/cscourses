Understanding Theta, Phi, and Lambda in CU1 and CU3 Rotation Matrices

Theta, phi, and lambda are angles used in the cu1 and cu3 rotation matrices. The cu1 rotation matrix is used to rotate a vector around the x-axis, while the cu3 rotation matrix is used to rotate a vector around the x, y, and z axes.


Theta is the angle of rotation around the x-axis in the cu1 rotation matrix. It is measured in radians and is the angle between the vector and the x-axis.


Phi is the angle of rotation around the y-axis in the cu3 rotation matrix. It is measured in radians and is the angle between the vector and the y-axis.


Lambda is the angle of rotation around the z-axis in the cu3 rotation matrix. It is measured in radians and is the angle between the vector and the z-axis.


The cu1 and cu3 rotation matrices are used to rotate a vector in three-dimensional space. The cu1 rotation matrix is used to rotate a vector around the x-axis, while the cu3 rotation matrix is used to rotate a vector around the x, y, and z axes.


The cu1 rotation matrix is defined as:


cu1(theta, ctl, tgt) = [cos(theta) -sin(theta) 0; sin(theta) cos(theta) 0; 0 0 1]


The cu3 rotation matrix is defined as:


cu3(theta, phi, lam, ctl, tgt) = [cos(phi) sin(phi) 0; -sin(phi) cos(phi) 0; 0 0 1] * [cos(theta) 0 -sin(theta); 0 1 0; sin(theta) 0 cos(theta)] * [cos(lam) sin(lam) 0; -sin(lam) cos(lam) 0; 0 0 1]
