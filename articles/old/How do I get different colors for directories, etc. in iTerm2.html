How do I get different colors for directories, etc. in iTerm2?

If you’re a Mac user, you’ve probably heard of iTerm2, the popular terminal emulator for macOS. It’s a great tool for developers, system administrators, and anyone else who needs to access the command line. One of the great features of iTerm2 is the ability to customize the colors of your directories, files, and other elements of the terminal. This can make it easier to navigate the command line and make your terminal look more visually appealing. In this article, we’ll show you how to get different colors for directories, etc. in iTerm2.


The first step is to open the iTerm2 preferences. You can do this by clicking on the iTerm2 icon in the menu bar and selecting “Preferences”. This will open the iTerm2 preferences window.


Once the preferences window is open, click on the “Profiles” tab. This will open a list of profiles that you can customize. Select the profile you want to customize and click the “Colors” tab. This will open the color settings for the profile.


In the color settings, you can customize the colors of the foreground, background, cursor, and other elements of the terminal. To change the color of a directory, select the “Directories” option from the list of elements. This will open a color picker where you can select the color you want for the directory.


Once you’ve selected the color you want, click the “Save” button to save your changes. You can repeat this process for any other elements you want to customize. When you’re done, click the “OK” button to save your changes and close the preferences window.
